== Description ==
Amazon Affiliate Reviews is a tool that adds a Reviews custom post if it does not exist and allows you to add Amazon product review to the Reviews section as well as add Amazon review to to WooCommerce products if WooCommerce is installed.

After you add your Amazon AWS Key & AWS Secret pair (needed to use Amazon AWS Service) to the plugin\'s Information page. This plugin when provided with an Amazon ASIN code & Amazon affiliate tag will retrieve an amzon product by providing the Amazon ASIN code.

The plugin will then if successful create 5 extra content areas and pre-fill 2 of them. all extra content areas are editable via the the admin menus for the reviews and if installed also WooCommerce product.

This plugin will also create a single sidebar widget for 5 popular Amazon product Reviews.

== Installation ==
1. Upload the `amz-free` folder to the `/wp-content/plugins/` directory
2. Activate the Amazon Affiliate Reviews plugin through the \'Plugins\' menu in WordPress
3. Configure the plugin by going to the `Information` menu that appears in your admin menu


== Frequently Asked Questions ==
1. Q. What are the requirement to add an Amazon product to the Reviews or WooCommerce Products main admin menu items? A. Your Amazon AWS Key ID & AWS Secret are both Required. 
2. Q. I have provided my Amazon AWS Key ID & AWS Secret, but still cannot get an Amazon product in. A. Make sure you provide the Amazon ASIN for each product.
